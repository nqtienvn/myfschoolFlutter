Kế hoạch code màn hình MyFschool

Bản kế hoạch có thể review cho các màn hình Flutter trong dự án. Bạn có thể duyệt phạm vi, chọn phương án, ghi chú từng phần, rồi xuất XML prompt ổn định cho bước code tiếp theo.

Ngữ cảnh đã đọc: lib/main.dart, màn login/home hiện tại, design system, repository/service layer, domain models, fixtures và tests. Các thay đổi sẵn có trong worktree được giữ nguyên.
